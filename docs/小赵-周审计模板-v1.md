# 小赵周审计模板 v1

用途：每周固定输出一次系统问题与训练建议，避免自由发挥写成长文。

输出规则：
1. 总字数尽量控制在 500 字内。
2. 只写本周新增或重复出现的问题。
3. 没有证据支持的猜测不要写进正式结论。
4. 每块最多 5 条。
5. 发现项如果已在 `小赵-问题库-v1.md`，优先引用编号，不重复展开背景。
