# 小赵每日自动汇报模板（老板版）

更新日期：2026-03-05
目标：每天固定产出“系统变化 + 高风险 + 员工遗漏”汇报，老板无需手动翻系统。

## 1. 执行命令（只读）

```bash
node --env-file=.env ops/codex/generate-daily-risk-report.mjs
```

## 2. Telegram 指令模板

```text
小赵，生成今日日报（只读）：
1) 今天系统发生了什么
2) 高风险清单（按严重度）
3) 员工可能遗忘未做事项（按人统计）
4) 明日优先动作建议
```

## 3. 固定输出结构

1. 今日关键数字
- 今日新增工单
- 今日异常流水（ADJUST / ROLLBACK）与大额DEDUCT
- 高优先未闭环工单
- 超时工单催办数（按负责人）
- 超时未点名催办数（课程结束超过3小时）

2. 高风险清单（Top）
- 财务高风险（ADJUST / ROLLBACK / 大额DEDUCT）
- 工单高风险（超时、高优先、按负责人）
- 教务高风险（超时未点名、按老师）

3. 员工遗漏清单
- 按工单 owner 的超时数
- 按老师超时未点名数

4. 明日优先动作
- 三条可执行动作，按先后顺序

## 4. 风控要求

1. 只读，不得写入。
2. 若数据源异常，明确标注“数据不完整”。
3. 对异常项必须给“责任人 + 截止时间”建议。
