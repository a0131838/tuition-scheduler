# Tuition Scheduler 员工使用手册（简版）

更新日期：2026-03-05  
口径：以 `docs/员工使用手册-完整版.md` 为准。

## 每天只做这 6 步

1. `/admin/todos` 清未点名
2. `/admin/conflicts` 清冲突
3. `/admin/schedule` 完成排课/调课
4. `/admin/students/[id]` / `/admin/classes/[id]/sessions` 补排课
5. `/admin/packages` + `/admin/packages/[id]/ledger` 核对课包与扣减
6. `/admin/receipts-approvals` + 财务报表完成当天审批/复核

## 岗位快速入口

- 前台：`/admin/students`、`/admin/booking-links`
- 教务：`/admin/schedule`、`/admin/conflicts`
- 学管：`/admin/packages`、`/admin/students/[id]`
- 财务：`/admin/receipts-approvals`、`/admin/reports/partner-settlement`、`/admin/reports/teacher-payroll`
- 老师：`/teacher`、`/teacher/availability`、`/teacher/sessions`

## 三条红线

1. 无有效课包不得排课。
2. 冲突必须在冲突中心闭环。
3. OpenClaw 小赵改数据必须 `Preview -> Confirm -> Apply`。
