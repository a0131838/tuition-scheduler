# Tuition Scheduler 员工简版手册（2页）

更新日期：2026-02-09
适用对象：前台/教务执行同事

## 1. 登录与语言
- 登录地址：`/admin/login`
- 点击：`进入管理端 / Admin`
- 右上角可切换语言：`双语 / 中文 / English`

## 2. 每日必做（按顺序）
1. 打开 `待办中心`（`/admin/todos`）
2. 处理 `昨日/今日未点名`
3. 处理 `明日提醒（老师/学生）`
4. 看 `冲突巡检`，有冲突就去 `冲突处理中心`
5. 看 `续费预警`，跟进将耗尽课包的学生

## 3. 常用页面
- `老师`：`/admin/teachers`
  - 维护老师资料、绑定账号、设置可上课时间
- `学生`：`/admin/students`
  - 新增学生、进入学生详情
- `学生详情`：`/admin/students/[id]`
  - 快速排课、查看出勤、导出课表 PDF
- `班级`：`/admin/classes`
  - 建班、进班级管理课次
- `报名`：`/admin/enrollments`
  - 新增/取消报名、导出报名 PDF
- `课时包`：`/admin/packages`
  - 创建/维护课包，检查余额
- `周课表`：`/admin/schedule`
  - 日常排课与改课
- `冲突处理`：`/admin/conflicts`
  - 换老师、换教室、取消课次/预约
- `学生选课链接`：`/admin/booking-links`
  - 创建链接、审批学生请求

## 4. 关键规则（必须遵守）
- 学生没有该课程有效课包，不能报名/排课。
- 同一学生同一课程不能重复创建报名关系。
- 同一学生可同时拥有：班课包 + 一对一时间包。
- 课包同模式有效期不能重叠（同学生同课程同模式）。
- 可上课时间限制：`08:00-23:00`。

## 5. 快速排课规则（学生详情）
- 只显示学生已购课程相关选项。
- 若无可用课包，会提示先建课包。
- 排课成功后弹窗自动关闭。
- 若没有可安排老师/课程，重新打开不会保留脏数据。

## 6. 导出功能
- 学生课表 PDF：学生详情页 `Download Schedule PDF`
- 报名 PDF：报名页导出
- 课包对账单 PDF：课包详情/对账单
- 选课链接 PDF：链接详情页导出
- 老师名片 PDF：老师名片页导出

## 7. 常见报错
- `Student has no active package for this course`
  - 先去课包页创建对应课程有效课包
- `No eligible teachers`
  - 检查老师授课科目、可上课时间、时间冲突
- `Outside availability`
  - 课次时间超出老师可上课时间

## 8. 遇到问题先做这三步
1. 刷新页面并重试一次
2. 回到列表页重新进入详情页
3. 记录截图 + 操作路径 + 时间点，提交给管理员
