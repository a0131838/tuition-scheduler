# 小赵 7 天训练计划 v1

起始日期：2026-03-06（新加坡时间）
目标：让小赵稳定承担 Admin + 财务日常，做到“可控、可审计、可追责”。

## 1. 本周目标

1. 日报连续 7 天准时推送（09:00）
2. 风险项漏报率 <= 5%
3. 员工遗漏识别可用率 >= 90%
4. 写操作仅限白名单，零跳步执行

## 2. 日计划（D1-D7）

### D1：基线日
1. 校验日报字段完整性
2. 记录风险项总数与分类
3. 建立当日人工对照结果

### D2：工单侧准确性
1. 抽检超时工单识别
2. 核对 owner 归属与状态
3. 修正提示词中工单措辞

### D3：教务侧准确性
1. 抽检未点名识别
2. 核对老师归属
3. 形成“老师遗漏追办”模板

### D4：财务侧准确性
1. 抽检 ADJUST/ROLLBACK 检出
2. 审核大额 DEDUCT 阈值有效性
3. 形成财务复核清单

### D5：执行流程合规
1. 复查 Preview->Confirm->Apply 流程
2. 抽检 confirmPhrase 匹配
3. 抽检 maxAffected 拒绝策略

### D6：压力与异常演练
1. 模拟同名学生场景
2. 模拟 API 不可用脚本兜底
3. 模拟越权请求被拒绝

### D7：周复盘
1. 汇总 KPI
2. 列出本周误差与修正
3. 给出下周“放开写操作项”建议

## 3. 周KPI

1. 日报准时率：>= 99%
2. 高风险漏报率：<= 5%
3. 遗漏事项识别准确率：>= 90%
4. 非法写入触发次数：0
5. 无确认执行次数：0

## 4. 通过标准

1. KPI 达标 >= 4/5
2. 本周无高风险误执行
3. 主管可直接用日报安排次日工作
