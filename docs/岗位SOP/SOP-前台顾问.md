# 岗位SOP：前台顾问

适用页面：`/admin/students`、`/admin/students/[id]`、`/admin/booking-links`

## 每日工作
1. 新增学生档案（姓名、学校、年级、来源、类型）。
2. 在学生详情核对是否有对应课程有效课包。
3. 需要学生自主约课时，创建 `学生选课链接` 并发送。
4. 跟进学生提交请求进度（提醒教务审批）。

## 标准操作
- 新增学生：`学生列表` -> `新增`。
- 查看学生详情：`学生列表` -> `编辑`。
- 创建选课链接：`学生选课链接` -> `创建链接`。

## 必查项
- 学生是否已购买目标课程课包。
- 选课链接的日期范围和失效时间是否正确。
- 链接是否复制并发给正确学生。

## 常见问题
- 报错 `Student has no active package for this course`：先让教务创建课包。
- 没有可匹配老师：联系教务确认授课老师和课程匹配。

## 交接输出
- 每日新增学生数
- 已发链接数
- 待审批请求数
