# 岗位SOP：学管（课包/续费）

适用页面：`/admin/packages`、`/admin/packages/[id]/ledger`、`/admin/students/[id]`、`/admin/reports/monthly-hours`

## 每日工作
1. 查看 `待办中心` 续费预警名单。
2. 在学生详情核对课包余额与近期排课。
3. 创建/更新课包，避免断课。
4. 对账：抽查课包流水与点名扣费一致性。

## 关键规则
- 学生可同时有班课包和一对一时间包。
- 同学生同课程同模式课包有效期不能重叠。
- 无有效课包不可报名/排课。

## 标准操作
- 新建课包：`课时包` -> `新增`。
- 查流水：课包列表进入 `ledger`。
- 导出对账单：课包导出 PDF。

## 风险信号
- 课包余额快速下降但排课量未明显增加。
- 同一学生多包重叠异常。
- 课包扣减与点名状态不一致。

## 每周输出
- 续费预警转化率
- 断课风险学生清单
- 对账异常清单
