# 岗位SOP：老师

适用页面：`/teacher`、`/teacher/availability`、`/teacher/sessions`、`/teacher/sessions/[id]`、`/teacher/card`

## 每日工作
1. 登录老师端，确认今日/明日课程。
2. 提前维护可上课时间（08:00-23:00）。
3. 每节课后完成点名。
4. 12小时内提交课后反馈。

## 标准操作
- 课程确认：`总览` -> 今日/明日确认按钮。
- 维护可用时间：`我的可上课时间`。
- 点名与反馈：`我的课次` -> 进入课次详情。
- 名片维护：`我的老师名片`，可下载PDF。

## 质量要求
- 点名状态准确（迟到/请假/缺勤不能误填）。
- 反馈内容具体（课堂表现、作业、前次作业完成）。
- 反馈逾期会进入告警。

## 常见问题
- 看不到课次：联系教务确认是否已排课/是否绑定老师账号。
- 无法提交反馈：先检查必填项和时间字段。

## 每周自检
- 反馈按时率
- 点名完整率
- 可上课时间是否覆盖下周
