# 小赵口语指令库 v2（老板专用）

更新日期：2026-03-05

## 1. 一句话查数据

1. `小赵，查学生“张三”的最近工单。`
2. `小赵，查学生“张三”最近30天课包流水。`
3. `小赵，查老师“李老师”明日冲突。`
4. `小赵，给我今天财务风险前三项。`
5. `小赵，看今天课表。`
6. `小赵，看今天课表（老师：李老师）。`
7. `小赵，看今天课表（校区：Bukit Timah）。`

## 2. 一句话做预览

1. `小赵，把“张三”最近工单里最新一条追加备注：已联系家长确认周三。先预览。`
2. `小赵，把“张三”最近流水中第1条备注改为：财务复核通过。先预览。`

## 3. 一句话确认执行

1. `小赵，确认执行：CONFIRM <operationId>`

说明：`confirmationText` 必须与 `confirmPhrase` 完全一致。

## 4. 每日节奏命令

1. 开工：`小赵，开工巡检。`
2. 中午：`小赵，中午复查。`
3. 收工：`小赵，收工复盘。`
4. 日报：`小赵，发今天的系统风控日报。`
5. 员工遗漏：`小赵，发今天员工遗漏追踪清单。`

## 4.1 问题与审计命令

1. `小赵，记录这个问题。`
2. `小赵，做本周系统审计。`
3. `小赵，更新这个规则到主文档。`

## 5. 小赵标准回报模板

1. 任务目标
2. 影响范围
3. 执行结果
4. 风险与回滚
5. 下一步建议

## 6. 失败兜底回复规范

1. 多个同名：返回候选编号让老板选 1 个
2. 无数据：只回 `无匹配学生` 或 `无匹配工单`
3. 无通道：提示可执行命令，不让老板手写 SQL

## 7. “看今天课表”固定执行规则（新增）

1. 默认解释：
- `看今天课表` = 查询今日（本地时区）可见课表，按老师端可见口径过滤请假。

2. 默认参数：
- `date=today`
- `includeExcused=false`
- `hideFullyExcused=true`

3. 调用优先级：
- 首选 API：`GET /api/admin/ops/daily-schedule-view`
- 兜底脚本：`node --env-file=.env ops/codex/query-daily-schedule-view.mjs "YYYY-MM-DD" "<teacherId可空>" "<campusId可空>"`

4. 固定返回字段：
- 课次时间（startAt-endAt）
- 老师
- 校区/教室
- 可见学生名单
- 汇总：`visibleSessions`、`visibleStudentCount`

## 8. 频率约束（新增）

1. `记录这个问题`：实时触发，但先过问题入库规则。
2. `系统风控日报`：每天 1 次，不重复输出历史内容。
3. `员工遗漏追踪清单`：每天 1 次，按当天待闭环项输出。
4. `本周系统审计`：建议每周 2 次（周三中午、周日晚）。
5. `更新这个规则到主文档`：只在老板明确批准后执行。
