# 两名教务培训计划（可执行版）

适用对象：Emily、Eva  
版本：v1  
依据文件：`/Users/zhw-111/Desktop/两名教务工作安排.docx`

## 一、培训目标

本培训的目标不是“知道流程”，而是让两名教务在固定分工下，能够稳定、独立、当日闭环完成以下工作：

1. 收集学生、家长、老师、合作方的排课与课程调整信息。
2. 准确判断事情属于：新排课、改课、补课加课、取消/请假、合作方问题、管理介入。
3. 每一条问题都录入工单，不漏单、不乱单。
4. 按工单推进，完成系统更新、回填结果、点击完成、归档。
5. 当天工单当天清；无法闭环的工单，要在当日升级或提醒管理。

## 二、岗位分工

### 1. Emily 的固定职责

Emily 负责前端收口和推进：

1. 收集所有消息。
2. 跟学生、家长、老师、合作方沟通。
3. 判断事情属于哪种工单类型。
4. 录入工单。
5. 跟进工单进度。
6. 把需要系统改动的内容完整交给 Eva。
7. 当工单卡住时，负责催办和升级。

一句话：`Emily 负责把事情收进来、讲清楚、推进出去。`

### 2. Eva 的固定职责

Eva 负责系统执行和结果闭环：

1. 按工单更新系统。
2. 修改排课、老师、班级、报名、课时包、备注等系统信息。
3. 更新后把结果回填到工单。
4. 能完成的工单点击完成并归档。
5. 发现信息不完整、逻辑冲突、系统无法执行时，立即退回 Emily 补充。

一句话：`Eva 负责把工单里的要求准确落到系统里。`

### 3. 两人共同责任

1. 任何事情先有工单，再有操作。
2. 不允许口头安排后不录工单。
3. 不允许改完系统不回填工单。
4. 不允许当天有未说明原因的遗留工单。

## 三、培训方式

培训分三段：

1. 规则培训：知道做什么、谁做什么。
2. 系统培训：知道点哪里、怎么录、怎么改。
3. 情景实操：按真实案例完整跑一遍。

培训周期：`5个工作日集中训练 + 5个工作日带教考核`

## 四、五天集中训练计划

## Day 1：角色和总流程

### 培训目标

让两人彻底明白分工和总流程。

### 培训内容

1. 讲清楚 Emily 和 Eva 的职责边界。
2. 讲清楚完整流程：
   - 收到消息
   - 判断问题类型
   - 录入工单
   - 推进沟通
   - 更新系统
   - 回填工单
   - 完成归档
3. 讲清楚什么情况必须管理介入：
   - 老师联系不上
   - 合作方投诉升级
   - 系统无法执行
   - 涉及费用争议
   - 同时影响多名学生/老师

### Day 1 实操

1. 用 5 个真实历史案例，口头判断工单类型。
2. 两人分别复述：
   - 我负责什么
   - 对方负责什么
   - 什么时候必须退回或升级

### Day 1 验收标准

1. Emily 能说清从“收到消息”到“工单交给 Eva”的完整步骤。
2. Eva 能说清从“收到工单”到“完成归档”的完整步骤。
3. 两人都知道不能跳过工单。

## Day 2：工单录入训练（Emily 重点）

### 培训目标

让 Emily 能独立、准确录入工单；Eva 能看懂工单并判断信息是否足够执行。

### 培训内容

1. 工单中心的用途。
2. 工单类型怎么选。
3. 优先级怎么选。
4. 状态怎么选。
5. `S – Situation` 三项怎么写：
   - 当前问题
   - 需要怎么做
   - 最晚截止时间
6. 学生、老师、课程、年级、微信群等字段什么时候必须填。
7. 证据上传怎么做。

### Day 2 实操

用 10 条真实情景训练 Emily 录工单：

1. 改课程时间
2. 改上课老师
3. 临时取消/请假
4. 补课加课
5. 新排课
6. 新学生购买课时包
7. 评估学生
8. 学术问题
9. 非学术问题
10. 合作方反馈问题

Eva 同时练习：
1. 看工单后判断“是否足够执行”。
2. 不足时要明确指出缺什么。

### Day 2 验收标准

1. Emily 连续录 5 张工单无漏字段。
2. `Situation` 三项完整。
3. Eva 对每张工单都能在 30 秒内判断“能否执行”。

## Day 3：系统更新训练（Eva 重点）

### 培训目标

让 Eva 能按工单更新系统；Emily 能核对是否改对。

### 培训内容

1. 学生页和排课入口的使用。
2. 改时间、改老师、补课加课、新排课怎么做。
3. 工单里需要的信息怎么映射到系统字段。
4. 什么时候不能直接改系统：
   - 工单信息不完整
   - 学生/老师不明确
   - 课时包不清楚
   - 时间/校区/老师冲突未确认
5. 改完以后如何回填工单。

### Day 3 实操

1. Emily 提交 8 张工单给 Eva。
2. Eva 按工单逐张更新系统。
3. Emily 负责复核：
   - 是否改对学生
   - 是否改对老师
   - 是否改对时间
   - 是否回填工单

### Day 3 验收标准

1. Eva 连续完成 5 张工单更新，不能错学生、错老师、错班级。
2. 每张工单改完都回填处理结果。
3. 两人都知道“信息不足时先退回，不硬做”。

## Day 4：合作与升级训练

### 培训目标

让两人学会协作，不互相推诿，不把问题挂过夜。

### 培训内容

1. Emily 交单给 Eva 的标准。
2. Eva 退回 Emily 的标准。
3. 什么叫“当天闭环”。
4. 什么叫“未完成但必须有原因”。
5. 每日收工前未完成工单怎么报管理。

### Day 4 实操

1. 做 6 个“会卡住”的案例：
   - 老师不回复
   - 家长没确认
   - 合作方信息不全
   - 系统找不到学生
   - 工单类型录错
   - 需要管理介入
2. 练习谁来补信息、谁来催、谁来升级。

### Day 4 验收标准

1. 两人都能说出自己何时该接、何时该退、何时该升级。
2. 遇到无法闭环工单，不会放着不管。
3. 每张卡住工单都能明确写出“卡点”和“下一步”。

## Day 5：整天模拟实战

### 培训目标

按真实工作节奏完整跑一天。

### 模拟内容

上午：
1. 新排课
2. 改时间
3. 补课加课
4. 老师无法上课

下午：
1. 合作方提出调整
2. 新学生购买课时包
3. 评估学生排课
4. 家长催进度

收工前：
1. 清当天工单
2. 列出未完成原因
3. 升级需要管理介入的项目

### Day 5 验收标准

1. 90% 工单当天闭环。
2. 其余 10% 必须有明确未完成原因。
3. 无漏工单。
4. 无未回填工单。

## 五、五天带教考核计划

## 第 2 周目标

从“会操作”变成“稳定操作”。

### 每天安排

1. 上午由两人独立处理真实工作。
2. 中午由管理抽查 5 张工单。
3. 下午继续处理。
4. 收工前复盘：
   - 哪些录错
   - 哪些改错
   - 哪些没升级
   - 哪些不该拖到第二天

### 每日考核指标

1. 漏工单数 = 0
2. 错工单类型 ≤ 1
3. 错改系统 = 0
4. 当日工单闭环率 ≥ 90%
5. 未完成工单必须有备注和升级记录

## 六、标准操作流程（傻瓜版）

## A. Emily 每天怎么做

### 早上开始

1. 打开消息。
2. 把所有新问题分类：
   - 改时间
   - 改老师
   - 补课加课
   - 新排课
   - 取消/请假
   - 新学生买课包
   - 评估学生
   - 合作方问题
3. 每一条都录工单。
4. 工单里写清楚：
   - 发生了什么
   - 要做什么
   - 最晚几点前做完
5. 能立刻补齐的信息先补齐。
6. 把需要系统执行的工单交给 Eva。

### 白天推进

1. 跟老师确认。
2. 跟家长确认。
3. 跟合作方确认。
4. 缺信息的立刻补。
5. 工单状态持续更新。

### 收工前

1. 检查今天所有工单是否都处理。
2. 已完成的，确认 Eva 已回填并归档。
3. 没完成的，写清：
   - 为什么没完成
   - 现在卡在哪里
   - 明天谁跟进
4. 需要管理介入的，单独发给管理。

## B. Eva 每天怎么做

### 早上开始

1. 打开工单中心。
2. 按优先级处理工单。
3. 先看工单信息是否完整。
4. 信息完整才开始改系统。

### 改系统时

1. 先确认学生对不对。
2. 再确认课程对不对。
3. 再确认老师、时间、校区对不对。
4. 改完后复查一次。
5. 把结果回填到工单。

### 做完后

1. 能完成的：点完成并归档。
2. 不能完成的：退回 Emily 补信息。
3. 涉及风险的：立即升级管理。

## 七、交接标准

Emily 交给 Eva 的工单必须具备：

1. 学生姓名明确。
2. 问题类型明确。
3. 需要修改的内容明确。
4. 最晚截止时间明确。
5. 必要证据已上传。

Eva 退回 Emily 的工单，必须明确写：

1. 缺什么信息。
2. 为什么不能改。
3. 需要谁去确认。
4. 最晚什么时候补回。

## 八、不能犯的错误

### Emily 不能犯

1. 收到消息不录工单。
2. 工单只写一句话，看不出要做什么。
3. 没确认清楚就催 Eva 改系统。
4. 工单卡住了也不升级。

### Eva 不能犯

1. 没工单就改系统。
2. 工单信息不完整还硬改。
3. 改完不回填工单。
4. 完成了不归档。
5. 改错学生、老师、时间、课程。

## 九、培训完成标准

两人培训完成，必须同时满足：

1. 连续 5 个工作日无漏工单。
2. 连续 5 个工作日无错改系统。
3. 工单当天闭环率达到 90% 以上。
4. 不能闭环的工单都能按规则升级。
5. 管理随机抽查 10 张工单，流程都说得清楚。

## 十、管理每天下班前检查什么

1. 今天是否有漏工单。
2. 今天是否有未回填工单。
3. 今天是否有已处理但未归档工单。
4. 今天是否有卡住但没升级的工单。
5. Emily 和 Eva 的交接是否清楚。

## 十一、建议实施方式

1. 第 1 周严格训练，不求快，只求正确。
2. 第 2 周边做边盯，重点纠错。
3. 第 3 周开始按正式岗位要求考核。
4. 第 4 周再决定是否放更多独立权限。
