# Eva 每日工作清单 v1

岗位定位：系统执行、结果回填、完成归档、异常退回/升级

## 一、上班后 30 分钟内

1. 打开工单中心。
2. 先看当天高优先级工单。
3. 按顺序整理待处理工单：
   - 紧急
   - 即将到期
   - 当天新建
   - 管理点名要先做的
4. 看每张工单信息是否完整。

## 二、执行前必须检查

1. 学生是否明确。
2. 老师是否明确。
3. 课程/科目是否明确。
4. 时间/校区/形式是否明确。
5. 工单要改什么是否写清楚。
6. 缺信息时不能硬做。

## 三、上午必须完成

1. 先处理紧急工单。
2. 先处理当天最早截止的工单。
3. 对能执行的工单，立刻改系统。
4. 对不能执行的工单，立刻退回 Emily。
5. 不允许把“看不懂的工单”挂着不动。

## 四、改系统时的顺序

1. 先确认学生。
2. 再确认课程。
3. 再确认老师。
4. 再确认时间。
5. 再确认校区/教室/授课形式。
6. 修改后复查一遍。
7. 确认没有改错对象。

## 五、改完后必须做

1. 回填工单处理结果。
2. 写明具体改了什么。
3. 如果没有完全完成，要写剩余卡点。
4. 如果已经处理完成，点击完成。
5. 完成后归档。

## 六、什么时候退回 Emily

1. 学生信息不完整。
2. 老师信息不完整。
3. 时间需求不明确。
4. 工单类型看起来不对。
5. 没有足够证据支持操作。
6. 财务/管理判断还没确认。

## 七、什么时候升级管理

1. 系统无法执行。
2. 同时影响多个班/多个老师/多个学生。
3. 涉及投诉、争议、责任判断。
4. 涉及权限外修改。
5. 工单逻辑相互冲突。

## 八、下午 3 点检查一次

1. 还有哪些高优先级工单没做完。
2. 有哪些已经退回 Emily 但还没补回。
3. 有哪些已改系统但还没回填工单。
4. 有哪些已完成但还没归档。
5. 有哪些工单可能今天收不了口。

## 九、下班前必须完成

1. 再看一次工单中心。
2. 已完成的全部回填并归档。
3. 未完成的全部写原因。
4. 需要管理介入的全部升级。
5. 不允许出现：
   - 改了系统没回填
   - 回填了但没归档
   - 卡住了但没人知道为什么

## 十、Eva 每天绝对不能漏的事

1. 没工单就改系统。
2. 工单信息不完整还硬改。
3. 改错学生、老师、课程、时间。
4. 改完不回填。
5. 完成不归档。
6. 卡住不升级。

## 十一、Eva 一天结束前自查

1. 今天是否有错改系统。
2. 今天是否有已改未回填工单。
3. 今天是否有已完成未归档工单。
4. 今天是否有未升级的异常工单。
5. 今天是否有能做却拖到明天的工单。
