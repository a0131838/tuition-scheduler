# Emily 每日工作清单 v1

岗位定位：信息收集、沟通协调、工单录入、推进、升级

## 一、上班后 30 分钟内

1. 打开所有消息来源。
2. 按来源检查新消息：
   - 家长
   - 学生
   - 老师
   - 合作方
   - 管理交办
3. 把需要处理的事情全部整理出来。
4. 判断每一条是否必须录工单。
5. 必须处理的事情，全部录工单。

## 二、录工单时必须检查

1. 学生姓名是否正确。
2. 老师姓名是否正确。
3. 课程/科目是否明确。
4. 问题类型是否选对。
5. 优先级是否选对。
6. 负责人是否明确。
7. `Situation` 三项是否完整：
   - 当前问题
   - 需要怎么做
   - 最晚截止时间
8. 截图/证据是否上传。

## 三、上午必须完成

1. 当天所有新消息都录入工单。
2. 高优先级工单先推进。
3. 需要老师确认的，上午先发出。
4. 需要家长补信息的，上午先发出。
5. 需要合作方确认的，上午先发出。
6. 把能执行的工单交给 Eva。

## 四、白天持续要做

1. 盯着老师回复。
2. 盯着家长回复。
3. 盯着合作方回复。
4. 工单状态要及时更新。
5. 发现资料不全，立刻补。
6. Eva 退回的工单，要立刻补信息再交回。

## 五、什么情况必须升级管理

1. 老师长时间不回复。
2. 家长投诉升级。
3. 合作方提出异常要求。
4. 工单超过截止时间还无法推进。
5. 涉及管理判断、责任争议、费用风险。
6. 同时影响多个老师或多个学生。

## 六、下午 3 点检查一次

1. 上午开的工单有没有都在推进。
2. 有没有工单没人跟。
3. 有没有老师还没回。
4. 有没有工单需要升级但还没升级。
5. 有没有当天高优先级工单快超时。

## 七、下班前必须完成

1. 再检查一次当天所有工单。
2. 已完成的，确认 Eva 已回填并归档。
3. 未完成的，必须写清：
   - 为什么没完成
   - 卡在哪里
   - 谁在跟进
   - 明天下一步是什么
4. 需要管理介入的，当天发给管理。
5. 不允许有“没人知道为什么没完成”的工单。

## 八、Emily 每天绝对不能漏的事

1. 有消息不录工单。
2. 工单类型选错。
3. Situation 写不完整。
4. 需要截图但没传。
5. 工单卡住了不升级。
6. 交给 Eva 的工单信息不完整。

## 九、Emily 一天结束前自查

1. 今天是否有漏工单。
2. 今天是否有未升级的卡点。
3. 今天是否有高优先级工单未跟进。
4. 今天是否有工单没有明确下一步。
5. 今天是否有已完成但未归档的工单没提醒 Eva。
