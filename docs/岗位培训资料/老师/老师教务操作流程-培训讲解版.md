# 老师 + 教务 操作流程（培训讲解版）

更新日期：2026-03-05  
建议时长：30-45 分钟  
口径来源：`docs/员工使用手册-完整版.md`

## 1. 培训目标

让新同事能独立完成：
- 建档
- 建课包
- 排课/改课
- 点名/反馈
- 财务复核

## 2. 讲解顺序

### 2.1 总控台（10 分钟）
- 演示页：`/admin/todos`、`/admin/conflicts`
- 结论：先风险后业务。

### 2.2 排课主流程（10 分钟）
- 演示页：`/admin/schedule`、`/admin/schedule/new`、`/admin/students/[id]`
- 强调：无有效课包不能排课。

### 2.3 点名与反馈（8 分钟）
- 演示页：`/admin/sessions/[id]/attendance`、`/teacher/sessions/[id]`
- 强调：点名与扣减要对得上。

### 2.4 财务闭环（8 分钟）
- 演示页：`/admin/receipts-approvals`、`/admin/packages/[id]/ledger`、`/admin/reports/partner-settlement`
- 强调：审批链完整后再导出。

### 2.5 工单流程（5 分钟）
- 演示页：`/admin/tickets`、`/admin/tickets/handover`、`/admin/tickets/archived`

## 3. 培训后考核

1. 从建档到排课完成 1 条闭环。
2. 能处理 1 条冲突。
3. 能解释 1 次扣减对账过程。
4. 能在老师端完成点名与反馈。
