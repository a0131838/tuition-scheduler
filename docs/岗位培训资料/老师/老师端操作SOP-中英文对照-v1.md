# 2. 老师每天上班先做什么
# 2. What a Teacher Should Do First Every Day

每天登录后，先按这个顺序处理：  
Every day after logging in, use this order:

1. 看 `Dashboard / 总览`  
   Open the dashboard first.
2. 看 `Sign-in Alerts / 签到告警`  
   Check sign-in alerts.
3. 看 `My Sessions / 我的课次`  
   Review your sessions.
4. 有当天课次就去做点名和反馈  
   If you have sessions today, complete attendance and feedback.
5. 收工前看 `My Payroll` 和 `My Expense Claims`（如果有需要）  
   Before ending the day, check `My Payroll` and `My Expense Claims` if needed.

# 8. My Availability / 我的可上课时间 怎么填
# 8. How to Fill in My Availability

这一页非常重要。老师如果没有及时更新可上课时间，教务就可能按旧时间排课，后面会出现：  
This page is critical. If a teacher does not update availability in time, admin may schedule based on old availability, which can cause:

1. 被排到你上不了的课  
   You may be scheduled into sessions you cannot teach.
2. 临时改课、改老师、补课冲突  
   Last-minute rescheduling, teacher replacement, or make-up conflicts.
3. 工单增加，影响家长体验  
   More tickets and a worse parent experience.

## 老师什么时候一定要更新
## When a Teacher Must Update Availability

出现下面任何一种情况，都要当天改，不要等教务来问：  
If any of the following happens, update it on the same day. Do not wait for admin to ask.

1. 这周某天临时不能上课  
   You become unavailable on a certain day this week.
2. 下周开始固定时间变了  
   Your regular schedule changes starting next week.
3. 考试周、学校活动、出差、请假  
   Exam week, school event, travel, or leave.
4. 原来能上的时间减少了  
   Your available hours become fewer.
5. 原来不能上的时间，现在又能上了  
   A previously blocked time becomes available again.

## 第一步：打开 My Availability / 我的可上课时间
## Step 1: Open My Availability

进入：`My Availability / 我的可上课时间`  
Go to: `My Availability / 我的可上课时间`

打开后你会看到 4 块：  
After opening the page, you will see 4 main areas:

1. `Quick Add (Single Day) / 快速添加（单日）`
2. `Bulk Add by Date Range / 批量添加（日期区间）`
3. `Availability Calendar (Next 30 days) / 未来30天可上课日历`
4. 如果你刚刚清空过某一天，还会看到 `Undo last clear-day / 撤回上次清空当天`

## 第二步：先判断你要改的是哪一种情况
## Step 2: Decide Which Type of Change You Need

### 情况 A：只改某一天
### Case A: Change One Specific Day

如果只是某一天新增一个可上课时间，用：  
If you only need to add availability for one day, use:

1. `Quick Add (Single Day) / 快速添加（单日）`

适合：
Suitable for:

1. 这周五临时可以上课  
   You are suddenly available this Friday.
2. 某一天多出一段时间  
   One specific day has an extra slot.

### 情况 B：一段日期都一样
### Case B: Same Pattern Across a Date Range

如果你接下来几周都差不多，用：  
If the same pattern applies for the next few weeks, use:

1. `Bulk Add by Date Range / 批量添加（日期区间）`

适合：
Suitable for:

1. 每周一到周五 16:00-20:00 都能上  
   You are available every Monday to Friday from 16:00-20:00.
2. 接下来一个月固定时段可上  
   You have fixed slots for the next month.

### 情况 C：某一天要删掉或改时间
### Case C: Remove or Edit One Day’s Slots

如果某一天原来填错了，或当天不能上了，用日历里的：  
If one day was entered wrongly or you are no longer available that day, use the calendar tools:

1. `Clear day / 清空当天`
2. `Manage raw slots / 管理原始时段`
3. `Edit / 编辑`
4. `Del / 删`

## 第三步：单日快速添加怎么填
## Step 3: How to Use Quick Add (Single Day)

在 `Quick Add (Single Day)` 填：  
In `Quick Add (Single Day)`, fill in:

1. 日期  
   Date
2. 开始时间  
   Start time
3. 结束时间  
   End time

然后点：  
Then click:

1. `Add / 添加`

操作例子：  
Example:

1. 你这周三 16:00-18:00 可以上课  
   You are available this Wednesday from 16:00 to 18:00.
2. 日期选本周三  
   Select this Wednesday.
3. 开始填 `16:00`  
   Enter `16:00` as the start time.
4. 结束填 `18:00`  
   Enter `18:00` as the end time.
5. 点 `Add / 添加`  
   Click `Add`.

## 第四步：批量添加怎么填
## Step 4: How to Use Bulk Add by Date Range

在 `Bulk Add by Date Range` 填：  
In `Bulk Add by Date Range`, fill in:

1. `From / 从`
2. `To / 到`
3. `Start / 开始`
4. `End / 结束`
5. 勾选星期几  
   Tick the weekdays you are available.

然后点：  
Then click:

1. `Bulk Add / 批量添加`

操作例子：  
Example:

1. 你从今天开始到四周后，每周一到周五 16:00-20:00 可以上课  
   From today until four weeks later, you are available Monday to Friday from 16:00 to 20:00.
2. `From / 从` 选今天  
   Set `From` to today.
3. `To / 到` 选四周后  
   Set `To` to four weeks later.
4. `Start / 开始` 选 `16:00`  
   Set `Start` to `16:00`.
5. `End / 结束` 选 `20:00`  
   Set `End` to `20:00`.
6. 勾选 `Mon / 周一` 到 `Fri / 周五`  
   Tick Monday to Friday.
7. 点 `Bulk Add / 批量添加`  
   Click `Bulk Add`.

## 第五步：怎么看日历
## Step 5: How to Read the Availability Calendar

下方 `Availability Calendar (Next 30 days)` 会显示未来 30 天。  
The `Availability Calendar (Next 30 days)` section shows the next 30 days.

你会看到：
You will see:

1. 每天的可上课时间段  
   The available time slots for each day.

   `No slots` when nothing is entered.

## 第六步：如果某一天临时不能上，怎么快速删除
## Step 6: How to Quickly Remove a Day if You Become Unavailable

如果某一天临时完全不能上课：  
If you are completely unavailable on one specific day:

1. 在日历里找到那一天  
   Find that day in the calendar.
2. 点 `Clear day / 清空当天`  
   Click `Clear day`.
3. 系统会把当天所有可上课时段清掉  
   The system removes all slots for that day.

这是最适合请假、考试、临时出差的做法。  
This is the fastest method for leave, exams, or travel.

## 第七步：如果你删错了，怎么撤回
## Step 7: How to Undo a Wrong Clear-Day Action

如果你刚刚点错了 `Clear day / 清空当天`，页面顶部会出现：  
If you cleared a day by mistake, the page shows:

1. `Undo available for last clear-day action / 可撤回的最近清空操作`

这时点：  
Then click:

1. `Undo last clear-day / 撤回上次清空当天`

注意：  
Important:

1. 这个撤回只针对最近一次清空当天  
   Undo is only available for the most recent clear-day action.
2. 不要删完很多天再想一起撤回  
   Do not clear many days and expect all of them to be undone together.

## 第八步：如果只是某一个时段要改，不想清空整天
## Step 8: How to Edit One Slot Without Clearing the Whole Day

如果一天里有多个时间段，只想改其中一个：  
If a day has multiple slots and you only want to change one:

1. 打开那一天下面的 `Manage raw slots / 管理原始时段`
2. 找到对应时段
3. 点 `Edit / 编辑`
4. 改开始和结束时间
5. 点 `Save / 保存`

如果这个时段完全不要了：  
If you want to remove only one slot:

1. 找到对应时段
2. 点 `Del / 删`

## 第九步：如果某一天要补一个额外时段
## Step 9: How to Add an Extra Slot Inside One Day

如果某一天本来已经有时间，但你想再加一段：  
If a day already has slots and you want to add one more:

1. 在该日历格里点 `Add slot / 添加时段`
2. 填开始和结束时间
3. 点 `Add / 添加`

适合：
Suitable for:

1. 晚上临时多出一小时  
   An extra hour opens up in the evening.
2. 周末某天新增一小段可上课时间  
   A weekend day gets one extra slot.

## 第十步：老师每天怎么用这一页
## Step 10: How a Teacher Should Use This Page Every Day

最推荐的习惯是：  
The recommended daily habit is:

1. 每周至少看 2 次  
   Check it at least twice a week.
2. 任何时间变化，当天就改  
   If your schedule changes, update it on the same day.
3. 不要等教务来追问才改  
   Do not wait until admin asks you.
4. 如果未来两周都固定，就直接用 `Bulk Add / 批量添加`
5. 如果只是临时变动一天，就用 `Clear day / 清空当天` 或 `Quick Add / 快速添加`

## 第十一步：老师最常见错误
## Step 11: Most Common Teacher Mistakes

### 错误 1：明明不能上，还忘记删
### Mistake 1: Forgetting to Remove Slots You Cannot Teach

后果：教务会按旧时间排课。  
Result: Admin schedules based on old availability.

正确做法：  
Correct action:

1. 当天马上进 `My Availability`
2. `Clear day` 或删掉具体时段

### 错误 2：一次性填太宽
### Mistake 2: Entering Overly Wide Availability

后果：你会被排到其实上不了的时间。  
Result: You may be scheduled into unrealistic time slots.

正确做法：  
Correct action:

1. 只填你真正稳定能上的时间

### 错误 3：直接口头说，不改系统
### Mistake 3: Telling Someone Verbally But Not Updating the System

后果：系统里的可上课时间还是旧的。  
Result: The system still shows old availability.

正确做法：  
Correct action:

1. 说完以后，自己马上进系统改

## 第十二步：一句话记住
## Step 12: One-Sentence Rule

老师端 `My Availability` 的原则只有一句：  
There is only one rule for `My Availability`:

1. **时间一变，当天就改；能批量就批量，改单天就改单天。**  
   **If your availability changes, update it the same day; use bulk add for recurring patterns, and single-day tools for one-day changes.**

# 16. 老师常见错误和正确做法
# 16. Common Teacher Mistakes and Correct Actions

## Mistake 1: Only checking the schedule but not taking attendance

正确做法：  
Correct action:

   Mark attendance as soon as possible after class.

## 错误 2：点名了但没交反馈
## Mistake 2: Attendance saved but feedback not submitted

正确做法：  
Correct action:

1. 点名后立刻交反馈  
   Submit feedback immediately after attendance.

## Mistake 3: Confirming payroll before understanding it

正确做法：  
Correct action:

1. 看不懂先暂停  
   Pause if unclear.
2. 先问教务或老板  
   Ask admin staff or the owner first.

## 错误 4：报销附件没检查就提交
## Mistake 4: Submitting reimbursement without checking the receipt

正确做法：  
Correct action:

1. 确保附件清楚  
   Make sure the receipt is readable.
2. 金额和附件一致  
   Ensure amount matches the receipt.

## 错误 5：看到告警但不处理
## Mistake 5: Ignoring alerts

正确做法：  
Correct action:

1. 告警出现就尽快处理  
   Handle alerts as soon as they appear.

   Do not wait until the owner or admin has to remind you.
