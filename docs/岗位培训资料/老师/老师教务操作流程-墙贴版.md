# 老师 + 教务 操作流程（墙贴版）

更新日期：2026-03-05  
口径：以 `docs/员工使用手册-完整版.md` 为准

## 教务每日五步

1. `/admin/todos` 清未点名
2. `/admin/conflicts` 清冲突
3. `/admin/schedule` 排课/改课
4. `/admin/students/[id]` 与 `/admin/classes/[id]/sessions` 补排课
5. 收工前复核待办与明日提醒

## 老师每日四步

1. `/teacher` 看课表
2. `/teacher/availability` 维护可上课时间
3. `/teacher/sessions/[id]` 点名
4. 12 小时内提交反馈

## 财务每日三步

1. `/admin/receipts-approvals` 审批
2. `/admin/packages/[id]/ledger` 核对扣减
3. `/admin/reports/partner-settlement` / `teacher-payroll` 复核

## 三条红线

1. 无有效课包不得排课
2. 冲突必须闭环
3. 小赵改数据必须 Preview -> Confirm -> Apply
