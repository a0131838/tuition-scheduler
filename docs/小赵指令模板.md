# 小赵指令模板（教务 / 财务 / 主管）

## 通用模板

```text
任务：<一句话目标>
对象：<表/页面/模块>
条件：<明确过滤条件>
变更：<字段A: old->new, 字段B: old->new>
限制：最大影响 <N> 条，不允许修改 <字段/范围>
模式：先 Preview，不得直接 Apply
```

## Apply 确认模板

```text
确认执行：CONFIRM <operationId>
同意按当前 preview 结果执行 apply。
```

## 教务模板

```text
任务：修正指定课次的老师分配
对象：session
条件：sessionId=<id>
变更：teacherId: <old> -> <new>
限制：最大影响 1 条；不得修改 startAt/endAt/classId
模式：Preview
```

## 财务模板

```text
任务：修正指定课包流水备注
对象：packageTxn
条件：packageId=<id> AND txnId in (...)
变更：note: <old> -> <new>
限制：最大影响 3 条；不得修改 deltaMinutes/kind
模式：Preview
```

## 主管模板

```text
任务：批量标记超时工单为已催办
对象：ticket
条件：status='Need Info' AND createdAt < <date>
变更：summary 追加 "[催办]"
限制：最大影响 20 条；不得修改 owner/status/createdAt
模式：Preview
```
